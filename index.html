IoT Gym

This was a self sponsored Capstone project where my team worked on developing a system that can be installed in commercial gyms, with an app to give users real time feedback on their workout with zero user input. Our approach to this was to use a system of cameras around the gym, then use machine learning techniques to determine what a person was doing and provide feedback and tracking.

In my fourth year project we developed a system that could automatically classify free weight exercises, count the number of reps performed and detect what weight was used. Data was collected using a Microsoft Kinect camera, and we implemented classification using a RNN, and developed signal processing algorithms for rep counting. Weight detection was done by looking at the color of weights the user was holding.

For my fifth year capstone, we developed calibration methods for a system of multiple Intel Realsense cameras. Our method allowed the relative positions of four cameras to be determined automatically, from videos of a calibration sphere being moved around the room. With this calibration, we could then generate a full reconstruction of a room, shown in the second image.

ENPH 253 Robot Competition

This was a class in my second year where we had to build robots that could navigate a mock city, picking up passengers along the way and dropping them off in a designated zone. On my team I was primarily responsible for our robot's code. I designed and implemented intersection detection algorithms to detect forks in the road as quickly and accurately as possible. We also successfully implemented a method for the robot to refind its position on the map if it got lost using rotary encoders. This ultimately paid off as we were the only team in the competetion to run completely autonomously through the whold competition, which we ended up winning.
